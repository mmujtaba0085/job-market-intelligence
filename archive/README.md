# archive/

Frozen, historical scripts and docs — one-off backfills, debug/diagnostic
tools, and point-in-time snapshots that are no longer part of active
tooling but are kept for reference.

Nothing here is maintained. Files may reference schema states, data, or
APIs that have since changed. If you need to resurrect one, read it
carefully before running it — don't assume it's still correct.

See [`../scripts/README.md`](../scripts/README.md) for the convention on
what belongs in `scripts/` versus here.
